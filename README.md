# Physics Simulations / Python & Pygame

A small collection of two independent physics simulations built with Python and Pygame.  
Each simulation can be run separately, or accessed from a simple main menu.

---

## Simulations 

### 1. Bouncing Ball
A ball that bounces on the ground and can be controlled using the Left and Right arrow keys. Debug informations are displayed.

- Reset  with R

### 2. Ball Explosion
A ball that falls from the top of the screen and explodes into several smaller balls upon impact.

- Randomized explosion patterns  
- Reset with R

---

## Main Menu

Run `main_menu.py` to open a simple interface with two buttons that launch each simulation.
